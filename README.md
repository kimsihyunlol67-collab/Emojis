# Emojis
Android Custom Emoji Keyboard App - Create, map, and use your own custom emoji layouts


# 🎨 이모지스 (Emojis)
> **안드로이드 커스텀 이모지 키보드 앱**  
> 사용자가 원하는 이모지를 원하는 키보드 위치에 직접 배치하고, 실제 키보드 입력 시 간편하게 스위칭하여 사용할 수 있는 앱입니다.

---

## 📌 주요 기능 (Features)

### 1️⃣ 트렌딩 및 상황별 이모지 추천 (Tab 1)
* 요즘 유행하는 이모지 조합 및 특수문자 피드 제공
* 클릭 한 번으로 간편하게 복사 및 보관함 추가

### 2️⃣ 나만의 이모지 컬렉션 (Tab 2)
* 자주 사용하는 이모지들을 카테고리별/폴더별로 관리
* 상황에 맞는 이모지 팩 커스텀 저장

### 3️⃣ 커스텀 키보드 설정 창 (Tab 3)
* **직관적인 그리드 배치:** 원하는 키 위치를 누르고 이모지를 검색/스크롤하여 자유롭게 배치
* **키 커스텀 관리:** 개별 키에 대한 이모지 추가 및 삭제 기능 제공
* **레이아웃 저장:** `Room Database` 기반으로 사용자가 설정한 키보드 배열 영구 저장

### 4️⃣ 실제 키보드 연동 (Android IME Service)
* **원터치 스위칭:** 시스템 자판 하단 바의 크기를 최적화하여 좌측 끝에 '이모지 전환 버튼' 배치
* **글자/이모지 빠른 전환:** 이모지 자판으로 이동 후 입력이 끝나면 다시 하단 버튼을 통해 기존 글자 키보드로 즉시 복귀 가능

---

## 🛠️ 기술 스택 (Tech Stack)
* **Platform:** Android
* **Language:** Java / Kotlin
* **UI Framework:** Android Architecture Components (Views, Bottom Navigation, RecyclerView)
* **Database:** Room / SharedPreferences
* **System Service:** InputMethodService (IME)
